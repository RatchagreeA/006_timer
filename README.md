# 25 + 5 Clock

## Overview
25 + 5 Clock is a web app that lets users set alarm timers for activity sessions and break sessions.  
(By default: activity session is 25 minutes and a break session is 5 minutes.)

## Built With
- Programming languages:
    - HTML
    - CSS
    - JavaScript

- Frameworks:
    - ReactJS
